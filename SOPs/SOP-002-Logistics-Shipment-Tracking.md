## 1. Purpose

This SOP establishes a standardized, secure process for managing 
and tracking shipments across the supply chain. It ensures operational 
efficiency, data integrity, and protection of sensitive logistics information.

---

## 2. Scope

Applies to all inbound and outbound shipments managed by the 
Operations & Logistics department.

---

## 3. Roles & Responsibilities

| Role | Responsibility |
|---|---|
| Logistics Coordinator | Create and manage shipment records |
| Operations Manager | Approve high-value shipments |
| IT Security | Ensure tracking systems are secured |
| Project Manager | Monitor timelines and escalate delays |
| Vendor/Carrier | Provide tracking updates and POD |

---

## 4. Security Considerations (CMP Framework)

- All shipment data stored in **encrypted systems only**
- Access restricted by **role-based access control (RBAC)**
- Vendor credentials managed through **secure key management**
- Any data breach involving shipment records reported within **24 hours**
- Third-party vendors must sign a **Data Handling Agreement**

---

## 5. Shipment Tracking Procedure

### Phase 1: Order Initiation
- [ ] Receive and verify purchase order
- [ ] Confirm vendor/carrier selection
- [ ] Generate shipment ID in tracking system
- [ ] Assign logistics coordinator to shipment
- [ ] Notify stakeholders via automated alert (Zapier)

### Phase 2: Pre-Shipment
- [ ] Verify inventory availability
- [ ] Prepare shipping documentation (BOL, packing list, customs forms)
- [ ] Confirm pickup date and time with carrier
- [ ] Log shipment details in dashboard
- [ ] Set automated tracking alerts

### Phase 3: In-Transit Monitoring
- [ ] Monitor real-time tracking updates daily
- [ ] Log any delays or exceptions immediately
- [ ] Communicate status updates to stakeholders
- [ ] Escalate critical delays to Operations Manager within 2 hours
- [ ] Document all carrier communications

### Phase 4: Delivery & Closure
- [ ] Confirm proof of delivery (POD)
- [ ] Inspect shipment for damage or discrepancies
- [ ] Update system to "Delivered" status
- [ ] File all documentation in secure storage
- [ ] Close out shipment record and archive
